# Sentiment Analysis Tool

A full-stack Sentiment Analysis Web Application that analyzes user input text and predicts sentiment (Positive, Negative, Neutral) using Machine Learning.

---

## How to Run Project

### 1. Clone Repository
git clone: https://github.com/sneha-r-r/sentiment-analysis-tool


### 2. Go to Project Folder
cd sentiment-analysis-app


### 3. Install Requirements
pip install -r requirements.txt


### 4. Run Server
cd backend
uvicorn main:app --reload
